Elliptic curves are symmetric over y=p/2y = p/2y=p/2

In secp256k1 over R2\mathbb{R}^2R2, the curve y2=x3+7y^2 = x^3 + 7y2=x3+7 forms an abelian group where point addition is defined as "draw a straight line through both points and take the inverse of the intersection (the third point)." The graph is symmetric along y=0y = 0y=0. There exists a point P∈R2P \in \mathbb{R}^2P∈R2 such that P=P−1P = P^{-1}P=P−1, namely P=(−73,0)P = (\sqrt[3]{-7}, 0)P=(3−7​,0). This point won't lie on secp256k1 if it is taken over (Fp)2(\mathbb{F}_p)^2(Fp​)2, so every point will have a distinct inverse in that case.

Plotting the curve over (Fp)2(\mathbb{F}_p)^2(Fp​)2

We vary x=0,1,2,3,…,p−1x = 0, 1, 2, 3, \ldots, p-1x=0,1,2,3,…,p−1 and substitute each into y2=x3+7mod  py^2 = x^3 + 7 \mod py2=x3+7modp. So for each xxx we get y2=qxmod  py^2 = q_x \mod py2=qx​modp, where qx=x3+7q_x = x^3 + 7qx​=x3+7.

If qxq_xqx​ is a quadratic residue of ppp, then the point lies on the curve. We get two solutions: (x,y)(x, y)(x,y) and (x,p−y)(x, p - y)(x,p−y). If yyy is even, then p−yp - yp−y is odd. Similarly, if yyy is a QR then p−yp - yp−y is a QNR. If qxq_xqx​ is a quadratic non-residue of ppp, then there is no solution.

The symmetry axis

The graph is symmetric along y=p/2y = p/2y=p/2 because yyy and p−yp - yp−y belong to the same xxx, since y2≡(p−y)2mod  py^2 \equiv (p - y)^2 \mod py2≡(p−y)2modp.

But wait. ppp is odd, so p/2p/2p/2 does not belong to Zp\mathbb{Z}_pZp​. Should we compute p⋅2−1mod  pp \cdot 2^{-1} \mod pp⋅2−1modp instead? That is always zero. But the EC curve is not symmetric over y=0mod  py = 0 \mod py=0modp.

Also, since ppp is odd, the number of quadratic residues and non-residues are not equal. But the yyy-coordinate points on the EC curve seem to be symmetric over y=p/2y = p/2y=p/2 anyway.

So the symmetry is visual, not algebraic in Zp\mathbb{Z}_pZp​. When we plot the points on a standard integer grid, yyy and p−yp - yp−y are equidistant from p/2p/2p/2 by construction. The axis y=p/2y = p/2y=p/2 is a property of how we draw the graph, not a meaningful element of the field.
