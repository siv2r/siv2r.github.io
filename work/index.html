BIP-445 — FROST Signing

BIP author·2024–presentfunded by Spiral (Block, Inc)

A specification for FROST signing on BIP340-compatible threshold Schnorr signatures, with support for BIP32 key derivation and BIP341 Taproot tweaking. I authored the BIP text, the Python reference implementation, and test vector suites covering every protocol algorithm — NonceGen, NonceAgg, Sign, DeterministicSign, PartialSigVerify, PartialSigAgg.

ChillDKG BIP

co-author, with Blockstream Research·2024–present

Schnorr Adaptor Signatures

contributor·2024

Rebased and extended the Schnorr adaptor signatures module in secp256k1-zkp, resolving pending review comments and adding a Multi-Hop Locks protocol example. Also authored Rust bindings in rust-secp256k1-zkp with type-safe Adaptor and SecretAdaptor wrappers.
