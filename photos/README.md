# Photos

Put 4 portrait JPGs here, named `portrait-1.jpg` through `portrait-4.jpg`.

Suggested specs:
- Aspect ratio roughly 4:5 (e.g. 400×490 px or 800×980 px)
- Quality ~80, file size under 200 KB each
- Variety — different outfits/backgrounds so the cycle feels alive, not repetitive

To add a 5th photo, save it as `portrait-5.jpg` and add a matching `<img>` element inside `.photo` in `/index.html`.

This README does not ship — it's just for you.
